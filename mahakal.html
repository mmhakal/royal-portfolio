Mahakal Developer Portal

Secure Admin & Employee Management Dashboard

Overview

The Mahakal Developer Portal is a robust, PHP-based admin and employee dashboard designed with a royal look and feel. It provides:

  • Secure admin login and employee login
  • Employee account creation and management
  • Profile updates with image & Aadhaar uploads
  • OTP-based password reset system
  • Separate dashboards for admin and employees

Key Features

Admin Dashboard

View and manage all employee activities in one place with an intuitive royal-themed UI.

Employee Profiles

Secure profile section where employees can update their info and upload documents.

Secure Auth

OTP-based password reset and session handling to ensure maximum security.

File Uploads

Easy upload of profile images and Aadhaar documents with built-in validation.
